# Store Screenshots

Raycast renders PNG files in this folder on the extension store page.

The current screenshots cover all three provider families:

- `ai-voice-studio-1.png`: MiniMax voice picker with 196+ multilingual voices.
- `ai-voice-studio-2.png`: MiMo voice picker with detail panel and text preview.
- `ai-voice-studio-3.png`: OpenAI voice picker with GPT-4o Mini TTS and recommended voices.
- `ai-voice-studio-minimax-*.png`: MiniMax long-form reading, voice selection, and clone-oriented workflow.
- `ai-voice-studio-mimo-*.png`: MiMo expressive TTS, TTS Studio, and voice-control workflow.

All screenshots are PNG files at 2000 x 1250 pixels.
